# PsylogBot

Телеграм чат-бот психолог для людей, которые нуждаются в психологической помощи, но по каким-то причинам не могут посетить специалиста. Также чат-бот может быть обычным собеседником. В экстренных случаях или по воле человека специалист горячей линии может войти в чат или позвонить. Результат может быть использован различными центрами психологической помощи.
